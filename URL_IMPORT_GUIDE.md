# URL导入功能使用指南

## 🎉 功能已上线！

URL导入功能已经完全集成到前端界面，你现在可以通过简单的界面操作导入任何公开网页的内容。

## 📍 访问方式

1. 打开浏览器访问: **http://localhost:3000**
2. 使用管理员账号登录:
   - 邮箱: `admin@admin.com`
   - 密码: `admin12345`
3. 在左侧菜单中点击 **"导入管理"**
4. 选择 **"URL快速导入"** 标签页

## ✨ 功能特性

### 1. 单个URL导入
- 输入任何公开网页的URL
- 可选择添加分类和标签
- 一键导入，自动提取内容
- 支持的格式：HTML → Markdown自动转换

### 2. 批量URL导入
- 一次导入多个URL（每行一个）
- 统一设置分类和标签
- 显示详细的导入结果
- 支持部分成功/失败的情况

### 3. 智能内容提取
- 自动识别文章标题
- 智能提取正文内容
- 过滤广告和无关内容
- 保留图片、代码块、列表等格式
- 提取作者、发布时间、标签等元数据

## 📝 使用步骤

### 单个URL导入

1. **输入URL**
   ```
   https://blog.example.com/my-article
   ```

2. **添加分类（可选）**
   ```
   技术文章
   ```

3. **添加标签（可选，用逗号分隔）**
   ```
   Python, 教程, 后端
   ```

4. **点击"立即导入"按钮**

5. **等待导入完成**
   - 成功后会显示文章标题
   - 自动跳转到知识列表

### 批量URL导入

1. **输入多个URL（每行一个）**
   ```
   https://blog.example.com/article1
   https://blog.example.com/article2
   https://blog.example.com/article3
   ```

2. **设置统一分类和标签（可选）**
   ```
   分类: 技术文章
   标签: Python, 教程
   ```

3. **点击"批量导入"按钮**

4. **查看导入结果**
   - 弹窗显示每个URL的导入状态
   - 成功/失败数量统计
   - 失败原因说明

## 🔧 故障排除

### 问题1: 导入失败
**可能原因:**
- URL无法访问（需要登录、付费等）
- 网站使用JavaScript动态加载内容
- 网络连接问题

**解决方案:**
- 确保URL是公开可访问的
- 尝试使用平台特定的适配器（CSDN、Notion等）
- 检查网络连接

### 问题2: 内容格式不正确
**可能原因:**
- 网站结构特殊
- HTML转Markdown时丢失格式

**解决方案:**
- 导入后手动调整格式
- 使用编辑器重新排版
- 反馈给开发团队改进

### 问题3: 导入速度慢
**可能原因:**
- 目标网站响应慢
- 内容较大（图片多）

**解决方案:**
- 耐心等待（最多30秒）
- 分批导入
- 避免高峰时段
