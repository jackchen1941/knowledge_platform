## 📝 测试步骤

### 方式1: 前端界面测试（推荐）

1. **打开浏览器**
   ```
   访问: http://localhost:3000
   ```

2. **登录系统**
   - 输入邮箱: `admin@admin.com`
   - 输入密码: `admin12345`
   - 点击"登录"

3. **进入导入管理**
   - 点击左侧菜单的"导入管理"
   - 选择"URL快速导入"标签页

4. **导入单个URL**
   - 在"文章URL"输入框中粘贴URL
   - 可选：输入分类（如：技术文章）
   - 可选：输入标签（如：Python, 教程）
   - 点击"立即导入"按钮
   - 等待导入完成

5. **查看导入结果**
   - 导入成功后会显示成功消息
   - 点击左侧菜单"知识管理"
   - 可以看到刚导入的文章

### 方式2: API文档测试

1. **打开API文档**
   ```
   访问: http://localhost:8000/docs
   ```

2. **登录获取Token**
   - 找到 `POST /api/v1/auth/login`
   - 点击"Try it out"
   - 输入登录信息
   - 点击"Execute"
   - 复制返回的 `access_token`

3. **授权**
   - 点击页面右上角的"Authorize"按钮
   - 输入: `Bearer YOUR_TOKEN`（替换YOUR_TOKEN）
   - 点击"Authorize"

4. **测试URL导入**
   - 找到 `POST /api/v1/import-adapters/import-url`
   - 点击"Try it out"
   - 输入参数:
     - url: 你要导入的URL
     - category: 分类（可选）
     - tags: 标签（可选，可以添加多个）
   - 点击"Execute"
   - 查看响应结果

## 🐛 常见问题

### 问题1: 导入失败
**可能原因**:
- URL无法访问
- 网站需要登录
- 网络连接问题

**解决方案**:
- 确认URL可以在浏览器中打开
- 使用公开可访问的URL
- 检查网络连接

### 问题2: 内容不完整
**可能原因**:
- 网站使用JavaScript动态加载
- 网站结构特殊

**解决方案**:
- 导入后手动补充内容
- 使用平台特定的适配器

### 问题3: 导入速度慢
**可能原因**:
- 目标网站响应慢
- 内容较大

**解决方案**:
- 耐心等待（最多30秒）
- 避免同时导入多个URL
